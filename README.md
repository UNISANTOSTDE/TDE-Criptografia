# Sobre o projeto

O projeto é um algoritmo de criptografia, que utiliza a cifra de Vigenere 

A cifra de Vigenère é um método clássico de criptografia que usa uma palavra-chave para cifrar mensagens
  
# Tecnologias utilizadas
## Back end
- Python
